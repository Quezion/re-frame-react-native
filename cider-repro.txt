https://github.com/clojure-emacs/cider/releases/tag/v0.24.0

=== SUCCESS PATH
Per the README.md, you'll need to run "npm install && cd react-native && npm install" from project root before this works.

To confirm that plain clojure CLI works at command line, navigate to project root in a terminal and type:
"clj -A:shadow-cljs watch app"

You should see the following output on a successful launch:
[:app] Compiling ...
[:app] Build completed. (133 files, 132 compiled, 0 warnings, 7.37s)
